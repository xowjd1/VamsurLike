인기 게임 뱀파이어 서바이벌을 모작한 2D 로그라이크 게임입니다.  

자동 전투 시스템과 다양한 무기/아이템 시스템을 구현했습니다.  

  
🛠 개발 환경  
  
  
엔진: Unity Engine  

개발 도구: Visual Studio  

개발 기간: 2024.03 ~ 2024.03  

개발 인원: 1명 (개인 프로젝트)
